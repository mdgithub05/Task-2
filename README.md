Superstore Sales Data Analysis using Excel (Pivot Tables & Insights)

The dataset contains the following key columns:
Order Date
Ship Date
Region
Category
Segment
Sales
Profit


Tools Used
Microsoft Excel
(Pivot Tables, Pivot Charts, Slicers, Conditional Formatting)

Tasks Performed
Cleaned the dataset (removed blanks, fixed date formats, corrected categories)
Converted raw data into Excel Table format
Created multiple Pivot Tables to analyze:
Total Sales by Category
Sales by Region and Segment
Profit Margin by Category and Region
Added Profit Margin as a calculated column
Applied conditional formatting to highlight top-performing regions
Created Pivot Charts for visual analysis
Added slicers for interactive filtering
Generated key business insights from the analysis
Exported the final report as PDF


Key Insights
Technology category generates the highest total sales.
West region is the top-performing region in terms of sales.
Consumer segment contributes the highest sales.
Furniture category has lower profit margins compared to others.
Central region shows potential for improvement.
